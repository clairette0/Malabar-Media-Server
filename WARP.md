# WARP.md

This file provides guidance to WARP (warp.dev) when working with code in this repository.

## High-Level Architecture

The Malabar-Media-Server is a portable, offline media server built on the ESP32-S3 microcontroller. The system is designed to be self-contained, creating its own Wi-Fi hotspot to serve media to connected devices without needing an internet connection.

### Core Components:

*   **Hardware:** The server runs on a Waveshare ESP32-S3 development board with a 1.47" LCD. All media is stored on a FAT32-formatted microSD card.
*   **Firmware:** The device's logic is managed by an Arduino-based firmware located in the `/firmware` directory. The main firmware file is `Nomad.ino`, which handles Wi-Fi setup, web server initialization, and system-level functions.
*   **Web Interface:** The user-facing interface is a set of HTML, CSS, and JavaScript files served directly from the microSD card. These files are stored in the `/SD_Card_Template` directory and should be copied to the root of the SD card.
*   **Media Indexing:** Media files (movies, music, books) are indexed in a `media.json` file, which is generated by the `media.py` script. This script scans the directory structure on the SD card and creates a JSON representation of the available media.
*   **Configuration:** Most settings, such as the admin password, RGB LED color, and Wi-Fi SSID, can be configured through the web-based admin panel. Initial Wi-Fi credentials are set in `firmware/Nomad.ino`.

## Common Commands

*   **Flashing the Firmware:**
    *   Use the Arduino IDE or PlatformIO to flash the firmware to the ESP32-S3. The main firmware file is located at `firmware/Nomad.ino`.

*   **Generating the Media Index:**
    *   To index your media files, run the `media.py` script. This will create the `media.json` file required by the web interface.
    *   `python3 SD_Card_Template/media.py`

*   **Editing the Web UI:**
    *   The web interface files are located in `/SD_Card_Template`. You can edit these files with any text editor to customize the UI.

*   **Editing the Screen UI:**
    *   The LCD screen interface can be modified using SquareLine Studio.
